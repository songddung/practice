### 가상 환경 생성
- python -m venv venv

### 가상 환경 활성화
- source venv/Scripts/activate

### 설치목록 확인
- pip list

### django 설치
- pip install django

### requirements.txt 생성
- pip freeze > requirements.txt

### requirements.txt
- pip install -r requirements.txt

### 프로젝트 생성
- django-admin startproject pjt2 .

### 앱 생성
- python manage.py startapp 앱이름

### 앱 등록
- setting.py > INSTALLED_APPS에 등록

### 앱 서비스 개발
- url 등록
- views 함수생성
- templates 폴더에 html생성
